# ng shop

Angular app with learning purpose

## Features

- ngrx state management
  - asyn with ngrx-effects

## Run'n'roll

### Environment

Install nodejs
```console
sudo apt install nodejs
```

Check nodejs
```console
node -v
```

Install npm
```console
sudo apt install npm
```

Check npm
```console
npm -v
```

Install Angular CLI
```console
sudo npm install -g @angular/cli
```

Check Angular CLI
```console
ng -v
```

### Project specific

Move to project folder

Instal npm packages
```console
npm install
```

Start server
```console
ng serve --open
```
